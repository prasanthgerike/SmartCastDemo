# SmartCastDemo

**SmartCastDemo** is an iOS app demo showcasing a native video streaming interface, optimized for Smart TVs and mobile devices. It demonstrates core iOS skills including Swift, UIKit, and REST API integration.  

## Features

- Native iOS App for iPhone and iPad  
- Integration with backend services using REST APIs  
- High-performance video streaming UI with responsive layouts  
- Smart TV interoperability (AirPlay 2, Chromecast)  
- Modular component design for reusability  
- Unit testing and basic debugging setup  

## Technologies Used

- **Languages:** Swift, Objective-C  
- **Frameworks:** UIKit, Cocoa Touch, Core Data, Core Animation  
- **Tools:** Git, Xcode, Instruments for performance tuning  
- **Architecture:** MVVM (Model-View-ViewModel)  
- **Version Control:** GitHub
